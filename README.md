Student Result Management System (SRMS)

Project Overview

The Student Result Management System (SRMS) is a web application that allows students to view their academic results by entering their matriculation number.
It automates result checking, reduces errors, and provides fast access to student scores and grades.

Features

1. Enter matric number to view results
2. Display student name, score, and grade
3. User-friendly interface

Technologies Used

1. HTML – Structure of the application
2. CSS – Styling and layout
3. JavaScript – Logic for displaying results
4. GitHub – Repository and deployment

Software Development Life Cycle (SDLC)

1. Requirement Analysis
 Identify system needs: view results, store student info, calculate grades
2. System Design
Design simple web interface with input box and result display
3. Implementation
Code written in HTML, CSS, and JavaScript
Folder structure:
SRMS/
├── index.html
├── style.css
├── script.js
└── README.md
4. Testing
Correct matric number displays result
Incorrect matric number shows “Result not found”
5. Deployment
Uploaded to GitHub
Can be accessed via GitHub Pages
6. Maintenance
Future improvements: admin login, database integration, better UI

How to Run the Project
1. Clone or download the repository
2. Open index.html in a web browser
3. Enter matric number 24/14360 to see the sample result
